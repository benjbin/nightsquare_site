# Images - Night Square

Ce dossier contient toutes les images utilisées sur le site Night Square.

## Structure recommandée

```
src/img/
├── hero/              # Images pour la section hero
├── events/            # Images des événements
├── cities/            # Images des villes/destinations
├── icons/             # Icônes et logos
└── organizers/        # Images pour la page organisateurs
```

## Formats recommandés

- **Format** : JPG ou WebP pour les photos, SVG pour les icônes
- **Optimisation** : Compresser les images pour le web
- **Résolution** :
  - Hero : 1920x1080px minimum
  - Événements : 800x600px minimum
  - Villes : 600x400px minimum

## Utilisation dans le code

Les images seront référencées ainsi :
- `src/img/hero/hero-background.jpg`
- `src/img/events/event-paris.jpg`
- `src/img/cities/paris.jpg`
